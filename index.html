FAQ

Answer the real questions. Do not pad this out with fluff.

How do I get access?

Create a Plex account first, then use the request access link on this page and include the email tied to your Plex account.

How do I know what is already available?

Use the library sheet before sending a request. It is the fastest way to see what is already on the server.

Can I request new content?

Yes. Requests are welcome. Check the sheet first, then send the request through the preferred contact path.

Do I need to donate?

No. Support is optional. If you choose to contribute, it should help with server costs and upgrades, not act like a paywall.
